# Account Book Web

## 기술 스택
- Vue 3
- Vue Router
- Pinia
- Composition API
- Vite

## 디자인
- Figma
- Bootstrap 5

## 기술 툴
- VS Code

## 브랜치 전략
- main: 최종본
- develop: 개발 통합본
- feature/*: 기능 개발

## 실행 방법
npm install
npm run dev

## 팀 규칙
- develop 브랜치에 직접 push하지 않는다.
- 모든 작업은 feature 브랜치에서 진행한다.
- 작업 완료 후 Pull Request 또는 팀원 확인 후 merge한다.
- 파일명과 컴포넌트명 작성 규칙을 통일한다.
- commit message 규칙을 통일한다.
